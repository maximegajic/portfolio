Application for micro-mobilities React Native

During my formation, I had the opportunity to develop an application. Our group was separated in two smaller groups : one for the back-end and one for the front-end. I was in the front-end one. We had to code in React Native and it was a new language for all of us. The goal of the application was to find a path from a place to another one, based on several criteria such as security, time… So we had to create a user interface for that purpose. This project was my first interface, and he helped me get to grips with design development

Alpha-beta-based AI implementation Python/C

As projects during my formation, I created two autonomous players, one of go and another of the Amazon board game. The first was in Python and the other in C. Both included alpha-beta approaches and a board heuristic as a strategy. Other techniques, such as opening libraries, were also used. These projects gave me a first idea of the techniques used for artificial intelligence and allowed me to practice on them

Board game C

It was also a good example for the use of header

Database management for a carpooling site SQL

CThis project, carried out during my training, involved creating and maintaining a database. The subject was school carpooling. Cryptography Python

This project, which I used as a TIPE during my CPGE studies, involved creating a cryptosystem based on the discrete logarithm problem in Python. As this was my first IT project, it allowed me to immerse myself in this domain and deepen my knowledge of Python
